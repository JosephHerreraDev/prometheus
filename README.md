# Prometheus

A modern arch rice based on [Omarchy](https://omarchy.org/).

## Installation

1. Clone in `~/.local/share/`
2. `cd prometheus`
3. `./boot.sh`.

> [!WARNING]
> This is only for a fresh arch linux install

## Features

- Hyprland window manager
- Theme switching
- Total personalization
